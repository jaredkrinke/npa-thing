How to generate input data

To create an excel report:
  1. Open flexible reporting
  2. Under "Admin" click "Appointments admin"
  3. Select desired start and end dates
  4. Set appointment status to "Active
  5. Select appointment types "Non-Service Payroll (Blue) Appointments" and "Service Appointments"
  6. Set show terminated staff to "No"
  7. Click on "Column Options"
  8. Select the following:
    • Appt. Status
    • Duration (Hrs)
    • Office Note
    • Result Note
    • Service Log Entry
    • Staff Job Title
    • Staff Name
    • Subject Line
  9. Click OK
  10. Click on "Run the report"
  11. Click on "Excel"
  12. Once your report opens, save as CSV (comma delineated)
